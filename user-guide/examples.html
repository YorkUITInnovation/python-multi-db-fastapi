Copy-paste ready examples for common tasks

Example 1: Creating a New User

Insert a new user record into the database:

curl -X POST http://localhost:8082/insertRecord \ -H "X-API-KEY: your_api_key" \ -H "Content-Type: application/json" \ -d '{ "dbtype": "mysql", "table": "users", "data": { "username": "alice_smith", "email": "alice@example.com", "firstname": "Alice", "lastname": "Smith", "created_date": "2024-12-06" } }'

Example 2: Finding a User by Email

curl -X POST http://localhost:8082/getRecord \ -H "X-API-KEY: your_api_key" \ -H "Content-Type: application/json" \ -d '{ "dbtype": "mysql", "table": "users", "parameters": {"email": "alice@example.com"} }'

Example 3: Updating User Status

curl -X POST http://localhost:8082/updateRecord \ -H "X-API-KEY: your_api_key" \ -H "Content-Type: application/json" \ -d '{ "dbtype": "mysql", "table": "users", "data": {"status": "active", "verified": true}, "where": {"email": "alice@example.com"} }'

Example 4: Paginated Query

Get the first 50 active users:

curl -X POST http://localhost:8082/sqlExec \ -H "X-API-KEY: your_api_key" \ -H "Content-Type: application/json" \ -d '{ "dbtype": "mysql", "sql": "SELECT * FROM users WHERE status = :status ORDER BY created_date DESC", "parameters": {"status": "active"}, "page": 1, "page_size": 50 }'
